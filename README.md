# stylelint-config-niksy

[![Build Status][ci-img]][ci]

[Stylelint][stylelint] config for my projects.

## Installation

```sh
npm install stylelint stylelint-config-niksy --save-dev
```

## Usage

Add this config to your `.stylelintrc`:

```json
{
	"extends": [
		"stylelint-config-niksy"
	]
}
```
